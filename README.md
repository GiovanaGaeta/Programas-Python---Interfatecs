# Desafios de Programação - InterFatecs 

Este repositório contém as minhas soluções para os problemas e exercícios propostos no curso Interfatecs e exercícios propostos na maratona. 
Todos os códigos foram desenvolvidos em Python.

## Sobre a Competição

A InterFatecs é uma competição voltada para estudantes das Fatecs de todo o estado de São Paulo.
